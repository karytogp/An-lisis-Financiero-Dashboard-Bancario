# 💰 Análisis Financiero / Dashboard Bancario | Power BI Project

## 📌 Descripción
Este proyecto consiste en un dashboard financiero enfocado en el análisis de productos, clientes y desempeño general de una entidad bancaria.

El proyecto está basado en una experiencia real, pero utiliza datos simulados para proteger la confidencialidad.

## 🎯 Objetivo
Analizar el comportamiento financiero y facilitar la toma de decisiones mediante indicadores clave.

## 🛠 Herramientas
- Power BI
- Excel
- SQL

## 📊 Análisis realizado
- Ingresos y egresos
- Segmentación de clientes
- Desempeño por producto
- Tendencias financieras
- Indicadores clave (KPIs)

## 📈 Resultados clave
- Identificación de productos más rentables
- Análisis de comportamiento de clientes
- Detección de patrones en ingresos y egresos
- Visualización clara para toma de decisiones

## ⚠️ Nota
Los datos utilizados en este proyecto son simulados y no corresponden a información real.
